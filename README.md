# React + Vite

College Event Manager

1. Setup Firebase project (Auth & Firestore)
2. Replace config in src/firebase.js
3. npm install
4. npm run dev

Flow:
- Clubs register as role 'club' -> can request a booking (status pending)
- Teachers review pending -> approve/reject
- Approved bookings appear on calendar and block that slot
- Students can view and register
```


 Features
- User Authentication (Students, Teachers, Clubs)
- Event Booking & Calendar Integration
- Role-based Dashboards (Student, Teacher, Club)
- Secure Routes with Firebase Auth
- Firestore Database Integration

---

 Tech Stack
- **Frontend:** React, Vite
- **Backend:** Firebase (Auth, Firestore)
- **Styling:** CSS
- **Hosting:** Firebase Hosting
